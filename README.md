测测你更像哪个文学家 v1.0.2
一款单页 H5 文学气质测试：不问典故、不考书目，凭雨夜、街角、话别与梦醒时的本能反应，从 66 位中外作家中找出与你气息最相近的那一位。

特点
66 位作家库：涵盖中国、日本、英美、法国、德语、俄语、拉美等地区文学家，附气质标签与名言摘记
22 道情境题：题目与选项偏文学化表达，弱化问卷感，约四五分钟完成
六维气质画像：现实批判、抒情浪漫、哲思抽象、都市孤独、人文悲悯、先锋实验
Top 5 匹配 + 雷达图：主匹配作家、近邻作家与可视化气质对比
答题专注模式：测试进行中隐藏介绍区与作家库，减少干扰
移动端友好：适配手机与微信 / QQ 内置浏览器，支持复制分享文案
纯静态单页：无后端、无登录，打开链接即可使用
在线体验
